# 麻雀 CLI

CLIで遊べる麻雀ゲームです。**ink**フレームワークを使用してターミナルUIを構築しています。

## 特徴

- ターミナルで動作する麻雀ゲーム
- 4人対戦（プレイヤー1名 + CPU 3名）
- Reactライクな記法でUI構築（ink使用）
- シンプルなゲームフロー

## 使用技術

- **TypeScript**: 型安全な開発
- **ink**: ターミナルUI構築フレームワーク（React for CLIs）
- **React**: inkの基盤

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発モードで実行
npm run dev

# ビルド
npm run build

# ビルド後に実行
npm start
```

## 遊び方

1. ゲームを起動すると、自動的に配牌されます
2. あなたのターンになったら：
   - **スペースキー**: 牌を引く
   - **← →**: 捨てる牌を選択
   - **Enter**: 選択した牌を捨てる
3. **Q**: ゲームを終了

## ゲームルール（簡易版）

- 各プレイヤーに13枚配牌
- ツモ→捨てるを繰り返す
- 現在は基本的な牌の引き捨てのみ実装
